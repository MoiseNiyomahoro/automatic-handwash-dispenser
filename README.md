# Automatic Hand Wash Dispenser

This project is a touchless **Automatic Hand Wash Dispenser** using **Arduino, Ultrasonic Sensor, LCD I2C, and LED**.  
The system detects a hand and automatically dispenses soap without physical contact.

## Components
- Arduino
- Ultrasonic Sensor (HC-SR04)
- LCD 16x2 (I2C)
- LED
- Water Pump / Servo

## Working
When a hand is detected by the ultrasonic sensor, the dispenser activates the pump and shows a message on the LCD.
